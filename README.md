# Fata_SMS
A simple python script for sms bombing. Sms bombing will only work in Bangladesh 

# First Time Installation 
$ pkg install git -y

$ pkg install python -y

$ pip install requests

$ pip inatall pytube

$ git clone https://github.com/XPL01T3R/Fata_SMS

# For Running
$ cd Fata_SMS

$ python fatasms.py
